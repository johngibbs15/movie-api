Overview

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Key Features

Endpoint Description Method URL Body Response
Returns a list of ALL movies to the user GET /movies None A JSON holding data about all movies
Returns data about a single movie by title to the user GET /movies/:title None A JSON object holding data about a single movie title
Returns data about a genre (description) by name/title GET /movies/genre/:genreName None A JSON object holding data about a single movie genre
Returns name of director GET /movies/directors/:directorName None A JSON object holding data about a single movie director
Allow new users to register POST /users A JSON object holding data about user to add A JSON obect with added user data
Allows users to update their user info (username) PUT /users/:id None An JSON object holding updated user data
Allows users to add a movie from their list of favorites POST users/:id/:movieTitle None A message indicating the movie title has been added
Removes a movie from a user's list of favorite movies DELETE /users/:id/:movieTitle None A message indicating the movie title has been removed
Allow existing users to deregister DELETE /users/:id/ None A message indicating user has been removed
